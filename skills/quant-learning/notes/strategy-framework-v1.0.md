## 一、理论假设 (Theoretical Foundation)

### 1.1 核心假设

**H1: 均值回归假设 (Mean Reversion)**
- **理论基础**: 价格在短期内围绕价值波动，极端偏离后会回归
- **数学表达**: P(t) = μ + ε(t), 其中 ε(t) ~ N(0, σ²), E[ε(t+Δt)] → 0
- **适用场景**: 震荡市场，高波动性资产
- **验证方法**: Hurst指数检验 (H < 0.5 支持均值回归)

**H2: 趋势持续性假设 (Trend Persistence)**
- **理论基础**: 价格存在动量效应，趋势会持续一段时间
- **数学表达**: 如果 P(t) > P(t-1) 且 P(t-1) > P(t-2)，则 E[P(t+1)] > P(t)
- **适用场景**: 牛市/熊市，宏观驱动行情
- **验证方法**: 自相关检验、动量因子测试

**H3: 波动率聚类 (Volatility Clustering)**
- **理论基础**: 高波动后倾向于高波动，低波动后倾向于低波动
- **数学表达**: σ²(t) 具有自相关性，可用 GARCH 模型建模
- **策略应用**: 根据波动率动态调整网格间距

### 1.2 市场微观结构假设

**H4: 订单流不平衡 (Order Flow Imbalance)**
- 大买单/卖单会推动价格偏离均衡，随后回归
- 网格策略利用这种偏离获利

**H5: 信息缓慢扩散 (Slow Information Diffusion)**
- 信息不会瞬间被全部市场参与者消化
- 价格调整存在滞后，产生可预测的波动

### 2.2 数学模型

#### 2.2.1 网格价格计算
```
Grid_i = Center_Price × (1 ± i × Grid_Spacing)

其中:
- Center_Price = EMA(20)
- Grid_Spacing = f(ATR) = k × ATR(14) / Price
- k: 调整系数，根据回测优化
```

#### 2.2.2 动态止盈
```
Take_Profit = Entry_Price × (1 + Grid_Spacing × TP_Multiplier)

TP_Multiplier 优化:
- 基于历史数据计算最优盈亏比
- 目标: 胜率 × 平均盈利 > (1-胜率) × 平均亏损
```

#### 2.2.3 凯利公式仓位
```
f* = (bp - q) / b

其中:
- f*: 最优仓位比例
- b: 盈亏比 (平均盈利/平均亏损)
- p: 胜率
- q: 败率 (1-p)

约束: 0.1 ≤ f* ≤ 0.5 (防止过度杠杆)
```

#### 2.2.4 趋势强度量化
```
ADX = 100 × |+DI - -DI| / (+DI + -DI)

趋势判断:
- ADX < 20: 弱趋势/震荡，适合网格
- ADX 20-40: 中等趋势，减少仓位
- ADX > 40: 强趋势，暂停网格或反向
```

## 三、假设验证方法 (Hypothesis Testing)

### 3.1 均值回归验证

**Hurst指数计算**:
```python
def hurst_exponent(ts, max_lag=100):
    lags = range(2, max_lag)
    tau = [np.std(np.subtract(ts[lag:], ts[:-lag])) for lag in lags]
    reg = np.polyfit(np.log(lags), np.log(tau), 1)
    return reg[0]  # Hurst指数
```

**判断标准**:
- H < 0.5: 均值回归 (支持网格策略)
- H = 0.5: 随机游走
- H > 0.5: 趋势性 (不适合纯网格)

### 3.2 参数稳健性检验

**Walk-Forward Analysis**:
1. 将数据分为 N 个周期
2. 每个周期用前一个周期优化参数
3. 验证参数是否稳定

**蒙特卡洛模拟**:
1. 对历史数据随机打乱
2. 测试策略在随机行情中的表现
3. 评估过拟合风险

## 四、性能评估体系 (Performance Metrics)

### 4.1 核心指标

| 指标 | 公式 | 目标值 | 理论依据 |
|------|------|--------|----------|
| **胜率** | Wins / Total | ≥80% | 行为金融学: 损失厌恶 |
| **盈亏比** | Avg_Win / Avg_Loss | ≥1.5 | 期望效用理论 |
| **夏普比率** | (R_p - R_f) / σ_p | ≥1.0 | 现代投资组合理论 |
| **最大回撤** | Max(Peak - Trough) | ≤30% | 风险管理 |
| **Calmar比率** | 年化收益 / 最大回撤 | ≥2.0 | 风险调整后收益 |
| **Kelly分数** | (bp-q)/b | 0.1-0.5 | 信息论/博弈论 |

### 4.2 统计显著性检验

**胜率显著性检验**:
```
H0: 胜率 = 50% (随机)
H1: 胜率 > 50%

检验统计量: z = (p̂ - 0.5) / √(0.25/n)
拒绝域: z > z_α (α=0.05, z=1.645)
```

## 五、优化框架 (Optimization Framework)

### 5.1 参数优化方法

**贝叶斯优化** (推荐):
```
目标: 最大化 Objective(胜率, 盈亏比, 夏普)
约束: 最大回撤 < 30%, 交易次数 > 100

```
遍历所有参数组合
选择满足约束的最优解

## 六、风险控制体系 (Risk Control)

### 6.1 事前风控

**市场状态过滤**:
- ADX > 40: 暂停策略 (极端趋势)
- ATR > 阈值: 减少仓位 (高波动)
- 隔夜跳空风险: 收盘前减仓

**仓位限制**:
- 单笔最大风险: 2% 账户资金
- 最大持仓: 50% 账户资金
- 凯利公式动态调整

### 6.2 事中风控

**止损规则**:
- 硬止损: 单笔亏损 > 3%
- 时间止损: N周期未盈利
- 波动率止损: 波动率突破时平仓

### 6.3 事后风控

**绩效归因**:
- 分析亏损交易特征
- 识别策略失效的市场状态
- 定期重训练模型参数
