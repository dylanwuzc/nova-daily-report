## 三、核心发现

### 发现1: 趋势过滤是双刃剑
- **严格过滤** → 高胜率，但错过大部分行情
- **宽松过滤** → 交易次数多，但胜率下降

### 发现2: BTC市场结构限制
- 50%时间为强趋势 (ADX>25)
- 网格策略只在震荡/弱趋势中有效
- 最多只能捕捉50%的交易机会

### 发现3: 参数存在最优权衡
- 无法同时满足: 胜率≥80% + 交易次数≥100
- 这是市场结构决定的，不是参数问题
